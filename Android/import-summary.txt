Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => Connect/src/main/AndroidManifest.xml
* assets/ => Connect/src/main/assets/
* res/ => Connect/src/main/res/
* src/ => Connect/src/main/java/
* src/siminov/connect/resource/DatabaseDescriptor.si.xml => Connect/src/main/resources/siminov/connect/resource/DatabaseDescriptor.si.xml
* src/siminov/connect/resource/ServiceRequest.si.xml => Connect/src/main/resources/siminov/connect/resource/ServiceRequest.si.xml
* src/siminov/connect/resource/ServiceRequestResource.si.xml => Connect/src/main/resources/siminov/connect/resource/ServiceRequestResource.si.xml
