
Siminov Connect is a lightweight, comprehensive, open source REST framework for the Android platform. It reduces the workload involved in building and consuming RESTful APIs in Android. 
It supports major Internet transport, data format, and service description standards like HTTP and HTTPS, SMTP, XML, and JSON.


Connect is divided into different components, each component is meant to execute specific tasks:

SERVICE:
	Service is a client-side communication component that process and handles any web service request.
	
NOFTIFICATION:
	It is a generic API which allows to handle push notification.
	
SYNC:
	It allows app to automatically checks for updates in the background, using battery and your data plan. 
	
AUTHENTICATION:
	Authentication is the process of determining whether someone or something is, in fact, who or what it is declared to be.
